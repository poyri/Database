fish

カラム名 データ型 値 内容（単位）
name varchar [文字列] 魚の名前
size varchar 1×1, 2×1 配置時の大きさ（マス）
price int [整数] 店での売値
place varchar 川, 池, 崖上, 河口, 桟橋, 海 釣れる場所
season varchar 一年中, ○月〜○月
例：5月〜12月		 釣れる時期（カッコ内は南半球）
time varchar 一日中, 朝/昼/夕/夜の組み合わせ（※1）
例：朝、夕、夜		 釣れる時間帯
weather varchar 雨天時のみ, NULL 天候条件
shadow varchar SS, S, M, L, LL, LLL, せびれ, ウツボ 魚影の種類
biting varchar 超長い, 長い, 普通, 短い, 超短い エサに食いついている時間。
短いほど難易度が高い。
m○
(○=1~12)		 int 0, 1 ○月に村で釣れるかどうか（北半球の時期）
（0：釣れない 1：釣れる）
t○
(○=0~23)		 int 0, 1 ○時に釣れるかどうか
（0：釣れない 1：釣れる）

※1 朝：4:00~8:59、昼：9:00~15:59、夕：16:00~20:59、夜：21:00~3:59
